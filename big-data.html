Hive Warehouse and Information Visualisation

Project Overview

The goal of this project was to develop a comprehensive data warehouse that could handle large-scale data processing and analysis, and to generate insightful visualisations to support data-driven decision-making.

Deliverables

  • Developed a scalable data warehouse using Hadoop and Hive, enabling efficient processing and analysis of vast datasets.
  • Optimised HiveQL queries to enhance the performance and reliability of the ETL (Extract, Transform, Load) processes.
  • Created impactful visualisations, including time series charts, heatmaps, radar plots, and bar graphs, to reveal key trends and correlations in the dataset.
  • Ensured data quality and coherence through rigorous data manipulation and cleaning using Pandas, leading to more accurate and meaningful visual outputs.

Technologies Used

  • Hadoop and Hive for building and managing the data warehouse
  • HiveQL for writing and optimising queries for data extraction, transformation, and loading.
  • Python for data analysis and visualisation
  • Matplotlib, Seaborn, and Plotly for developing detailed and interactive data visualisations.
  • Pandas for data manipulation and cleaning, ensuring high data quality before visualisation.
